Потрачено времени на изучение технологий - неделя.

Чистого времени на решение задачи - неделя.

Сложности - долгая компиляция и выбрасывание множества разнообразных эксепшенов.




1. Установите MySQL.

2. Клонируйте репозиторий.

2. Откройте его как проект в Intellij IDEA

3. в application.properties измените имя пользователя и пароль на данные вашего MySQL-сервера.

4. Запустите Application.java

5. Зайдите в браузер по адресу localhost:8080.
